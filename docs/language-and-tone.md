# Language & Tone

## Labels
- UI labels are concise and action-oriented: "Add keyword" not "Click here to add a new keyword"
- Button labels are verbs: "Save", "Export", "Delete" — not "OK" or "Submit"

## Feedback Messages
- Error messages explain what happened and what to do: "Failed to load data. Try refreshing the page."
- Empty state messages are helpful, not apologetic: "No keywords added yet. Add one to get started."

## Terminology
- Avoid jargon in user-facing strings
- Use SEO/marketing terms only when the audience is clearly technical
